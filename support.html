Common Questions

How do I connect my Bybit account?

Go to Settings → Connect Bybit. Enter your API Key and Secret. Make sure your API key has "Read" and "Trade" permissions enabled on Bybit. We recommend creating a dedicated API key with IP restriction for maximum security.

Is my data safe?

Yes. Your API keys are stored in the iOS Keychain (hardware-encrypted). All analysis runs on your device. Nothing is sent to our servers — we don't have any. See our Privacy Policy for details.

What exchanges are supported?

Currently, MeatGrinder supports Bybit USDT Perpetual Futures. More exchanges may be added in future updates.

The app shows "Not configured"

This means no Bybit API keys are connected. Market data (prices, analysis) works without API keys. Portfolio and trading features require a connected account.
